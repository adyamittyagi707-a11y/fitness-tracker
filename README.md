🏋️ Fitness Tracker

A simple fitness tracker web app built using HTML, CSS, and JavaScript. 
Users can add exercises along with their duration and view them in a list.

🚀 Features:
- Add workouts
- Display workout list
- Simple and clean UI

🛠️ Tech Used:
- HTML
- CSS
- JavaScript

📚 Learning Outcome:
- DOM manipulation
- Form handling
- Dynamic content rendering
